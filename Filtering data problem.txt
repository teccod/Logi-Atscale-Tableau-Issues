В таблицу выводятся данные из двух полей через display fields (страны и значение, разбивающееся по этим странам). На таблице стоит фильтр по 3-ему полю вида 3-е_поле like какая_то_строка (в данном случае по версии продукта). Предполагается видеть вывод вида значение фильтруется по 3-ему полю, потом разбивается на страны. Что получаем - сначала значение разбивается по странам, потом фильтром like фильтруется, и для каждой версии продукта появляется своя запись страна-значение. В выводе таблицы наблюдаем повторение названия страны с цифрами для разных версий, хотя группировки по ним в chart wizard не предполагалось.
